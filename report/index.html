Glass Solid Viewer

We wish to create an application that accurately depicts a glass material on a number of different models.
This includes correct diffuse/specular lighting for semi-transparent to transparent materials, as well as the reflection and refraction that are indicative of glass.
Functionality includes being able to change basic properties of the glass material such as color and transparency, being able to rotate the camera around object, and being able to rotate the object itself.


*This feature will use the three.js library


User Features:
View glass solids on a fully-rendered display.
Toggle through different preset glass models.
Interact with the application to rotate the models.
Full 360 degrees of camera rotation.
Change the color and transparency of each glass model.
